# Credit Card Generator

A simple and elegant web application built with vanilla JavaScript that generates random credit card information.

## Features
- **Random Data**: Generates credit card numbers, expiry dates, and CVVs.
- **Modern UI**: A clean and responsive design for easy interaction.
- **Fast & Lightweight**: Built with standard web technologies (HTML/CSS/JS).

![Preview](./snips/snip.png)

## Quick Start
1. Clone the repository:
   ```bash
   git clone https://github.com/hussainsulehri/Creadit_Card_Generator.git
   ```
2. Open `index.html` in your browser.

## Built With
- **HTML5**: Structure and semantics.
- **CSS3**: Modern styling and layout.
- **JavaScript**: Core logic and random data generation.
